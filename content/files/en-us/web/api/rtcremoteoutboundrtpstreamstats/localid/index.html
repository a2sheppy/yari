{{APIRef("WebRTC")}}

The {{domxref("RTCRemoteOutboundRtpStreamStats")}} dictionary's  localId property is a string which can be used to identify the {{domxref("RTCInboundRtpStreamStats")}} object whose {{domxref("RTCInboundRtpStreamStats.remoteId", "remoteId")}} matches this value. Together, these two objects provide statistics about the inbound and outbound sides of the same synchronization source (SSRC).

Syntax

let localId = rtcRemoteOutboundRtpStreamStats.localId;

Value

A {{domxref("DOMString")}} which can be compared to the value of an {{domxref("RTCInboundRtpStreamStats")}} object's {{domxref("RTCInboundRtpStreamStats.remoteId", "remoteId")}} property to see if the two represent statistics for the inbound and outbound sides of the same stream.

Example

In this example, statistics reports are collected before and after a network operation in order to determine how many packets are being sent and received by that operation each time it's executed. The network operation itself is handled by a function called doSomeNetworkActivity(), which returns a promise that resolves when the network activity is completed.

async function testNetwork(pc) {
  let sender = pc.getSenders()[0];

  let firstReport = await sender.getStats();
  await doSomeNetworkActivity();
  let secondReport = await sender.getStats();

  for (let endStats of secondReport.values()) {
    if (endStats.type === "outbound-rtp") {
      let beginStats = firstReport.get(endStats.id);

      if (beginStats) {
        let remoteStart = firstReport[beginStats.remoteId];
        let remoteEnd = secondReport[endStats.remoteId];

        let packetsSent = endStats.packetsSent - beginStats.packetsSent;
        let packetsReceived = endStats.packetsReceived - beginStats.packetsReceived;

        console.log(`Sent ${packetsSent} packets; received ${packetsReceived}.`);
      }
    }
  }
}
